Базовая грамматика арифметических выражений:

```
E -> E + T | E - T | T
T -> T * F | T / F | F
F -> V^F | V
V -> ( E ) | id | num | - V
```

***

Грамматика арифметических выражений без левой рекурсии, после левой факторизации:

```
E  -> T E'
E' -> + T | - T | ""
T  -> F T'
T' -> * F | / F | ""
F  -> V F'
F' -> ^F | ""
V  -> ( E ) | id | num | - V
```
